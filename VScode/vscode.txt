# Remote-SSH
前提：
ローカル端末からリモート端末にSSH接続できること

手順：
*****
ただし、「~/.ssh/config」を設定している場合はかなりの手順を無視できる
*****
1. 拡張機能「Remote-SSH」を導入する
2.「表示」メニューから「コマンドパレット」を開き「remote ssh」と検索する
3. 検索結果の「Remote-SSH: Add New SSH Host･･･」を選択する
4. 画面左側のメニューに「リモートエクスプローラー」が追加されるのでこれを開き、
   SSH接続先の追加を行う
5. 追加されたホストが表示されている項目の右にあるフォルダのようなボタンを押すと接続が開始される
* SSHのパスワードを求められる
6. 通常通りに「フォルダーを開く」などから作業開始

## 便利な拡張機能 ##
####################
# Japanese Language Pack for Visual Studio Code
VSCodeの言語を日本語に変換する拡張機能

# vscode-icons 
アイコンによってフォルダやファイルの種類が人目でわかりやすくなる拡張機能

# Code Spell Checker
英単語などのタイプミスを教えてくれる拡張機能

# zenkaku
全角と半角のスペースの違いを表示してくれる拡張機能
全角スペースは黄色くハイライトされる

# Path Autocomplete
パスを途中まで入力すると自動で補完してくれる拡張機能
Pathの予測を出してくれる

# indent-rainbow
インデントを色付けしてくれる拡張機能
Pythonなどを書くときに便利

# Atom One Dark Theme
目に優しい配色にしてくれる拡張機能
かつての神コーディングアプリ「Atom」のような色合いにしてくれる

# Trailing Spaces
コードの後ろについている無意味なスペースがわかりやすくなる拡張機能
無意味なスペースを赤色でハイライトしてくれる

# Rainbow CSV
CSVファイルのそれぞれの要素がどのカラムに属するかを色付けしてくれる拡張機能

# Remote Development
Dockerコンテナ内でのデバッグなどができる
