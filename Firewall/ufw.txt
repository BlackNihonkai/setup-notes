###############################
## ufwのよく使うコマンドたち ##
###############################

________________________________________________________________________________________________________________________
|              |                   |
|  ufw allow   |  ルールを追加     |  後ろにポート番号を指定してルールを追加する
|  ufw enable  |  ufwを有効化      |  ルールが設定されていない限り、新規接続ができなくなるので注意
|  ufw disable |  ufwを無効化      |
|  ufw status  |  ufwの状態を表示  |  後ろに「numbered」をつけて使うと良い
|  ufw delete  |  ルールを削除     |  後ろにポート番号や、「numbered」で表示された項目番号を書いてルールを削除する
|______________|___________________|____________________________________________________________________________________

##########################
## 指定アドレスとポート ##
##########################

_________________________________________________________________________________________________________________________________________
|                                                                         |
|  ufw allow from ネットワークアドレス to ホストアドレス port ポート番号  |  ネットワークアドレスからホストアドレスへの通信を指定
